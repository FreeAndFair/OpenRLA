About

Quick Sampler is an open-source web application for efficiently and transparently choosing items at random from a collection.

It does this by running the SHA-256 pseudo-random sampling algorithm described publicly in 2011 by Ronald L. Rivest. The application can be used for things like selecting precincts at random for a post-election manual audit.

For more information about Quick Sampler and Rivest's algorithm, including the source code for this application, visit the Quick Sampler project website on GitHub.
